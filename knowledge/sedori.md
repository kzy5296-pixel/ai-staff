## リサーチ手法

### 店舗せどり

**初心者〜月5万円までの優先ルール：値下げワゴン特化**
- 迷ったらまず「値下げワゴン」だけ見る。通常棚の細かい値下げ札リサーチは、月5万円達成後でよい。
- 店に入ったら全通路を一度歩き、レジ前・通路沿い・店舗奥・隅のワゴンを探す。
- 店舗ジャンルは、家電量販店・ホームセンター・ディスカウントショップから始める。
- 見込み利益と実際の現金は別物。仕入れ量を増やす前に、売れる速度と支払い日を確認する。

**ドンキホーテ攻略メモ**
- 仕入れ前準備：majicaアプリ、majica donpen card、店舗LINE登録、クーポン確認。
- 狙う値札：驚安大放出、圧倒的驚安、しくじり市、特売、値下げしました、破格値、赤字覚悟の大処分、倉庫市。
- 初心者は「広告の品」は優先度低め。全国的に同価格で競合が増えやすい可能性がある。
- 狙いやすいカテゴリ：理美容家電、調理家電、おもちゃ、家庭用雑貨。
- PB（プライベートブランド）はAmazon出品不可・出品しづらいことが多いので原則スルー。
- 箱潰れ・日焼け・破れが強い商品は新品出品時の返品・クレームリスクが高い。
- 縦積み前に、Keepaで出品者数急増がないか、他店舗でも同じセールか、店員に全国セールかを確認する。
- 配送無料サービスがある店舗は、大型商品や車なし仕入れで使える。FBA用ダンボール・緩衝材確保にもなる。

**家族時間・信頼貯金の注意**
- 楽天ポイントせどりや大量電脳仕入れは、セール日にスマホ/PCへ張り付きやすく、荷物到着も増える。
- 育児中は「利益額」だけでなく、家のスペース、荷受け負担、家族の睡眠・静かな時間もコストとして見る。
- カズヤさんは3児育児中なので、低単価大量仕入れより「短時間で確認できる店舗ワゴン」「少量高利益」「在庫が家を圧迫しない商品」を優先する。

**中古せどりの使いどころ**
- 新品せどりで資金が詰まりやすい場合、中古は1個あたり利益が大きく、少量仕入れで利益を作れる可能性がある。
- Googleレンズ、メルカリ販売履歴、型番検索で相場確認する。
- 最初は「売れなくても自分で使える」「損しても勉強代で済む」低リスク商品から試す。
- 家具・大型品は利益が大きい一方、梱包・発送の難易度が高い。初心者は梱包方法まで確認してから仕入れる。

### 電脳せどり：刈り取り（Amazon）

**刈り取りとは：**
Amazon 1P（Amazon直販）が一時的に大幅値下げした商品を仕入れ、在庫切れ後に通常価格に戻ったタイミングで3P FBAとして売る手法。

**理想の刈り取り条件（全部揃って初めて買い）：**
1. ✅ Amazon 1P（販売元：Amazon.co.jp）が出品中
2. ✅ 現在価格が90日平均より20%以上安い（delta ≥ 20%）
3. ✅ 現在価格が90日間の最安値（isLowest90）
4. ✅ 他のECサイト（楽天・Yahoo等）でも在庫がない or 大幅に高い ← **超重要**
5. ✅ 3P出品者が少ない or 競合がいない
6. ✅ 評価4.0以上（返品リスク低減）
7. ✅ 売れ筋ランキングが良い（全体3,000位以内が理想）
8. ✅ 小型〜中型サイズ（≤2kg、≤450mm）でFBA手数料を最小化

**⚠️ 条件4が最重要：** 楽天など他サイトで安く在庫がある商品は、Amazon在庫切れ後も3Pセラーが他サイトから仕入れて安値で出品してくるため価格が回復しない。

**🛒 Amazonカート（Featured Offer）抑制の仕組み：**
Amazonは楽天・Yahoo!ショッピング・ヨドバシ等、複数の外部サイトの価格を監視している。
自分の出品価格が外部サイトの最安値より高い場合、カートが表示されず「すべての出品を見る」にのみ表示される状態になる。
→ カートが出ない = 購入までのクリック数が増え、売れにくくなる。
→ 外部サイト全体で自分の価格が最安 or 同等になると、カートが復活する可能性がある。

**対応策：**
- 外部サイトの最安値を下回る価格に設定してカートを取りに行く
- または外部サイトの在庫が売り切れるのを待つ（楽天等の在庫量次第）
- 刈り取り判断時に「楽天採算NGかどうか」を見るのは、3P参入しにくさだけでなくAmazonカート取得のしやすさにも直結している

**📦 仕入れ後の出品方法：自己出品で待機が基本**
刈り取りは「Amazon在庫切れ → 価格回復」のタイミングを狙う手法。
FBA納品すると倉庫に届くまで数日〜1週間かかるため、価格が動いた瞬間に間に合わない可能性がある。
→ **価格回復の見通しが立つまでは手元在庫で自己出品し、価格が上がったタイミングで即出荷できる体制を維持する。**
（価格が安定してきた・ロット数が多い場合はFBA切り替えも検討）

**刈り取りの5W1Hメモ：**
- What：何の商品を狙うか。型番・JAN・バリエーション違いまで確認する。
- Why：なぜ価格が下がっているか。単発セール、在庫処分、廃盤、トレンド終了など理由を考える。
- When：いつ仕入れて、いつ売るか。値上げ前・在庫切れ前後・季節需要の前を意識する。
- Where：Amazonだけでなく、楽天・Yahoo!・店舗・メーカー公式など市場在庫を見る。
- Who：誰が買う商品か。需要が明確な商品ほど価格回復後も売れやすい。
- How：自己出品待機、FBA投入、寝かせ、少量テストなど出口を決めてから仕入れる。

**トラッキング型の考え方：**
- 刈り取りは「その場で見つける」より、候補商品を大量に登録して値下がり通知を待つ運用が強い。
- 最初から1,000件登録を狙うと挫折しやすいので、まずは小型・出品制限が少ないカテゴリで100件程度から始める。
- 値下がり通知が来たら、価格差だけでなく、他EC在庫・Amazon在庫・3P競合・出品可否を再確認する。
- LEGOや大型家電のような場所を取る商品だけでなく、小型で保管負担が少ない商品リストも作る。
- トラッキングは「登録して終わり」ではなく、通知後のメンテナンスが重要。通知が弱い場合は、候補商品の質・条件・登録カテゴリを見直す。
- 有料記事内の自動登録系ツールはWindows前提のものがある。カズヤさんはMac運用なので、同じ考え方をKeepa・QuickShop・自作リストで代替する。

**狙い方のバリエーション：**
- 値上がり前：メーカー値上げ・価格改定・セール終了前に仕込む。
- トレンド：テレビ・YouTube・SNS・メディア露出で需要が跳ねた商品を確認する。
- 季節：夏物・冬物など毎年需要が戻る商品は、オフシーズン安値で候補化する。
- 廃盤：生産終了・型落ち・旧モデル化で市場在庫が減る商品を追う。
- 他EC刈り取り：Amazonだけでなく楽天・Yahoo!側の一時値下げも候補にする。

**バリエーション商品の確認ポイント：**
- アパレル・白衣・下着などはサイズ/色/型番違いでライバルが分散しやすい。
- ただし、サイズ違いごとに回転が違うため、親商品全体ではなくバリエーション単位で見る。
- 確認軸：評価、販売データ、在庫、レビュー、サブカテゴリー順位。
- 粗利率が高く見えても、サイズ違いの需要が薄いものは寝かせリスクがある。
- レビュー数は総数だけでなく、直近で増えているかを見る。総レビューが多くても最近増えていなければ、今は動いていない可能性がある。
- 薄くて軽いアパレル系は、超安値で拾えた場合にメルカリ等の逃げ道も作りやすい。ただしサイズ・色ごとの回転確認が必須。

**鉄板条件と妥協ライン：**
- 最強条件：Amazon価格が40〜50%前後下落、ランキング3桁、楽天・Yahoo!など他ECが同じように値下がりしていない。
- 妥協ライン：ランキング4桁、40%前後下落、他ECが値下がりしていない。
- 他ECが安く残っている商品は、Amazon在庫切れ後も他セラーが仕入れられるため価格回復が鈍い。ここは妥協しない。
- 廃盤確度が高い商品は、値下がり幅がやや小さくても検討余地あり。ただし廃盤確認が弱いなら通常の鉄板条件を優先する。

**廃盤確認ルート：**
- ヨドバシ等の商品ページで「販売終了」表示を確認する。
- Amazonの商品名・商品ページで「旧モデル」「新しいモデルがあります」表示を確認する。
- コスメ系は@cosme等で「生産終了」「リニューアル」情報を確認する。
- メーカー公式サイト内検索で、現行ページが残っているか、生産終了ページに移っているかを見る。
- 判断に迷う商品はメーカー問い合わせまで検討する。1つの情報だけで廃盤認定しない。

**他EC刈り取りの深掘り：**
- Keepa通知が来てAmazonで買えない場合も、JAN・型番・キーワードで楽天・Yahoo!・メーカー系ショップ・専門店を確認する。
- 現行品は市場在庫が厚いと価格が戻りにくいので、できるだけ安く仕入れる。
- 他ECの納期が2〜3か月など長い場合は、実在庫が薄いサインになることがある。
- 楽天・Yahoo!はポイント込みでAmazon仕入れより条件が良くなることがある。ただしポイントを利益に入れすぎない。
- Amazon/Rakuten/Yahoo!で在庫が薄い時は、PC専門店・卸系サイトなどのニッチ在庫も確認対象にする。

**値上げ前・トレンド・季節の考え方：**
- 値上げ前刈り取りは「安くなった通知」だけでなく、Keepaの価格上昇通知も使う。価格が上がり始めたら、まだ安く残っている他EC在庫を探す。
- トレンドはテレビ・YouTube・SNSで急に動くが、供給が多い商品は回復も早い。短期売り切り前提で見る。
- 季節商品は夏物を冬〜春、冬物を1〜2月以降に確認するなど、需要期の前に仕込む。
- 年末商戦は9〜10月から候補確認を始める。12月に探し始めると、すでに仕込みが遅い場合が多い。

**カズヤさん向け優先順位：**
1. 廃盤・限定品 × 小型 × 他EC値下がりなし
2. 値上げ前通知 × 小型 × 出品制限なし
3. 店舗ワゴン品を、Amazon刈り取りと同じ基準で確認
4. アパレル系は利益率が高い一方、バリエーション確認が重いので慣れてから少量テスト

## Amazon売れ筋起点リサーチ

**考え方：**
- 「安いものを探す」より先に、「Amazonで実際に売れているもの」を起点にする。
- FBA開設後に見られるAmazon出品大学の「週ごとの推薦商品」レポートを使う。

**手順：**
1. セラーセントラルでAmazon出品大学を開く。
2. 「レポートとデータを活用しよう」→「週ごとの推薦商品」を確認。
3. 任意カテゴリの日本語Excelを保存。
4. JANコードで楽天・Yahoo!ショッピング・店舗在庫を検索。
5. Amazon価格、Keepa波形、他EC在庫、出品制限を見て仕入れ判断する。

**注意：**
- ランキング上位でも利益が出るとは限らない。売れている商品リストは「候補リスト」であり、最終判断は価格差・回転・競合・出品可否で行う。
- 楽天リアルタイムランキングは「流行確認」には使えるが、同時に仕入れ競合が増えやすいサインとして逆方向にも見る。

## 資金管理・キャッシュフロー

**最重要：利益管理と現金管理は別物**
- 利益が出ていても、仕入れ代金の支払い、Amazon売上金の入金タイミング、在庫滞留で現金が減ることがある。
- 「見込み利益があるから仕入れる」ではなく、「支払い日までに現金化できるか」を見る。

**仕入れ前チェック：**
- 今月の仕入れ予定額
- 今月のカード支払い予定額
- Amazon/メルカリ等からの入金予定額
- 手元現金
- 在庫金額と滞留期間
- 追加仕入れしても生活費・固定費に影響しないか

**やってはいけないこと：**
- 売れる速度より速く仕入れ続ける
- 利益額だけ見て縦積みする
- 資金ショートをリボ払い・分割払いで埋める
- 家の在庫スペースを超えて仕入れる

**カズヤさん向け運用：**
- まずは月の仕入れ上限額を決める。
- 1商品あたりの想定利益だけでなく、何日で現金化できそうかもメモする。
- せどりの成果報告では「見込み利益」「実利益」「手元現金」「在庫残」を分けて見る。

### グリッドからデータ抽出するコード

```javascript
// ページをロード（1ページずつ呼ぶ）
const gridEl = document.querySelector('.ag-root-wrapper');
const api = gridEl.__agComponent.gridOptions?.api || gridEl.__agComponent.api;
api.paginationGoToPage(1); // 0始まり。呼ぶたびにforEachNodeの件数が増える

// データ抽出・フィルタリング
const rows = [];
api.forEachNode(n => { if(n.data) rows.push(n.data); });
const amazonKW = ['amazon','fire tv','echo ','kindle','fire hd','fire stick'];
const results = rows.map(d => ({
  asin: d.asin,
  title: (d.title || '').substring(0, 50),
  price: d.AMAZON_current,
  rank: d.SALES_current,
  delta90: d.AMAZON_deltaPercent90,
  avg90: d.AMAZON_avg90,
  weight: d.packageWeight,
  brand: (d.brand || '')
}))
.filter(r => !amazonKW.some(k => r.title.toLowerCase().includes(k) || r.brand.toLowerCase().includes(k)))
.sort((a, b) => b.delta90 - a.delta90);
```

### AMAZON_deltaPercent90 の意味（重要！）
- **正の値** = 現在価格が90日平均より低い（値下がり中）→ **買いシグナル**
- **負の値** = 現在価格が90日平均より高い（値上がり中）→ 買わない
- 計算式：`(avg90 - current) / avg90 × 100`

## FBA手数料の目安

| カテゴリ | 販売手数料 | FBA配送料（目安） |
|---|---|---|
| DVD・映像 | 15% | ¥350〜600 |
| DIY・工具・ガーデン | 8% | ¥400〜600 |
| 楽器・音響 | 8% | ¥500〜800 |
| ホーム＆キッチン | 10% | ¥400〜700 |

**簡易粗利計算：**
`粗利 = 売値 - 仕入れ値 - (売値 × カテゴリ手数料%) - FBA配送料`

## 仕入れ基準・ルール

- 小型〜中型サイズ優先（≤2kg、≤450mm）でFBA手数料を抑える
- 評価3.5以下は返品リスクが高いので避ける
- 予約商品はisLowest90に引っかかることがある → 必ずAmazonで発売済みか確認
- 出品制限に注意：Apple製品はほぼNG、アディダス・ナイキ等のブランド靴もNG多い
- **靴系ブランド全般（アシックス・ニューバランス・メレル・プーマ・キーン・ミズノ等）は出品制限リスク高め → スクリーニング段階で除外ワードに入れる**
- CERO-Z（18禁ゲーム）は出品設定が複雑 → 初心者は避ける。Amazonページを開くとアダルト警告が出る
- ミュージック（CD）カテゴリは出品制限がかかりやすい → 要申請
- P&Gブランド（ブラウン等）も出品制限リスク高め
- **3P他出品者が0件の商品は出品制限の可能性がある → 候補から外す**
- 他出品者が1件以上いる商品 = 出品できる実績あり → 優先して検討
- **「入荷予定あり」の商品は注意** → Amazonが継続して安値供給する可能性があり、在庫切れが前提の刈り取りでは不利
- **シリアルコード・メーカー特典付きゲームは注意** → 流通品のコードが使用済みの可能性あり

### Keepa グラフ各ラインの正しい読み方

| ライン | 意味 |
|---|---|
| **ピンク線** | BuyBox / カート取得価格。相場価格を確認する基準 |
| **グリーン線** | ランキング推移。売れると数値が下がり一気に折れる。折れた回数 = 売れた個数。「バキバキ」= よく売れている |
| **オレンジ線** | Amazon本体の出品価格。オレンジ塗りつぶし = Amazon出品期間 |
| **水色線（最下段）** | 新品アイテム数（出品者数）の推移 |

**★ チャンスシグナル：ピンク線の上にオレンジ線がある状態**
= Amazon が出品しているのにカートを取れていない商品 → 競合の仕入れを諦めさせやすい好条件

**出品者数増加 → 価格競争・値下がり予想。減少 → 価格回復の可能性**

### 仕入れ判断チェックリスト（10ステップ）

新品せどり全般に適用できる実践フレームワーク。電脳刈り取りでも共通して使える。

1. **Amazonカタログに登録されているか確認**（ASIN/JAN検索）
2. **利益が取れるか**（目標利益率 15〜20%以上が目安）
3. **出品制限はないか**（セラーアプリで確認。Request = 現状出品不可）
4. **相場価格の確認**（カート価格 ≠ 相場価格。直近3ヶ月の「実際に売れた価格帯」を見る）
5. **新品が売れているか確認**（中古メインで動いている商品は新品では売りにくい）
6. **バリエーション確認**（Keepa の Ratings 数でどのバリエーションが売れているか判定）
7. **ライバル数確認**（最悪ケース = 直近3ヶ月の最大出品者数で割った個数が月1以上あるか）
8. **カート取得者確認**（Amazon の Cart 取得率が 80% 以上なら基本回避）
9. **真贋調査リスクの波形チェック**（出品者数が突然急減している ＝ 真贋調査の可能性）
10. **直近の価格・出品者数のトレンド確認**（下落途中・出品者増加中なら見送り）

**★ 最悪のケース = 値下げしても売れないこと（損切りすらできない）**
→ どんな状況でも「直近3ヶ月の最低価格で売っても赤字にならないか」を必ず確認

---

### 真贋調査リスクを避けるサイン

以下の特徴がある商品は真贋調査リスクが高い → 回避推奨

- 出品者数に「崖」がある（一気に激減している時期がある）
- 出品者数が常に 1〜2 件で安定している（価格もほぼ固定）
- メーカー本体 or 正規店のみが出品している商品
- カート取得率で Amazon が 80% 以上を占めている

---

### カート価格と相場価格の違い（重要な落とし穴）

**カート価格（現在の最安値）≠ 市場の相場価格**

例：カート価格 ¥2,500 でも、実際に売れている価格帯が ¥1,500 であれば、¥2,500 では売れない。
→ Keepa のピンク線を見て「どの価格帯でランキング線が折れているか（= 売れているか）」で相場を判断する。

---

### バリエーション確認の方法（Keepa Ratings 活用）

Keepa の Ratings 数（レビュー数）はバリエーション別の売れ行きに比例する。

例：カラーが5種類ある商品で「ブラック」のレビューが全体の8%しかないなら、ブラックは全体の8%しか売れていない。
→ 手元のバリエーションが本当に売れているかを確認してから仕入れる。

---

### Keepa Product Finder のマイナス入力（逆張り検索）

`deltaPercent` などのフィールドに `-100` のようにマイナス値を入力すると **逆の意味（値が上がっているもの）** になる。

例：`AMAZON_deltaPercent90` に `-100` と入力
→「90日平均より現在価格が 0〜100% 上がっている商品」を抽出

→ 需要増加・在庫減少で価格が上昇中の商品を見つけるのに使える。

また **Related Best Sellers** 機能で、お宝商品の周辺ジャンルの類似商品も一括でリサーチできる。

### FBA 倉庫での紛失・破損補償申請方法

FBA倉庫では頻繁に紛失・破損が発生する。定期的にチェックして申請することで損失を回収できる。

**手順：**
1. セラーセントラル → レポート → フルフィルメント
2. 「在庫元帳」→ ダウンロードで表示
3. レポートタイプ「詳細表示」、イベントタイプ「調整」、期間「365日」でCSV取得
4. CSV の N列「未照会数量」が 1 以上 = 紛失の可能性がある FNSKU
5. ヘルプ → サポートとリソースを利用する → 「私の問題は掲載されていません」
6. 「在庫調整レポート」と入力 → FNSKU をコピペ → 送信
7. Amazonから「補填しました」または「補填対象外」の返事が来る

**定期的に（月1回程度）確認する習慣をつけると損失を防げる。**

### 廃盤・限定品の価格特性

- **廃盤予定・生産終了商品**: 需要が続くのに供給が止まる → 価格が上昇しやすい

- **季節外れ品**: 春に冬物ヒーターを仕入れ → 秋冬に高く売る（季節差益）
