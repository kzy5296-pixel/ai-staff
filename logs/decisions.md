# 重要判断・方針変更の履歴

> 重要な決断・方針転換・大きな気づきを日付つきで記録する。
> 「なぜその判断をしたか」の文脈も一緒に残す。
> 後から振り返ったときに意思決定の経緯が追えるように。

---

## フォーマット

```
### YYYY-MM-DD — タイトル
- **判断内容**：
- **背景・理由**：
- **期待する結果**：
- **振り返り**：（後日追記）
```
