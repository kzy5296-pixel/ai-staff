## 追加・変更された機能

### 本日（05-04）の変更

**bilateral フラグ最終修正**
- インクラインカール・インクラインハンマーカール・ダンベルカール・3wayダンベルカール → `bilateral: true` を削除（両手同時種目のため片側記録不要）
- ワンアームケーブルカール・コンセントレーションカール・ワンアームチンニング → `bilateral: true` を維持（正しく片側記録が必要）

### 05-03 の主要変更（継続参照）

| 機能 | 概要 |
|------|------|
| ホーム「今日のおすすめ」カード | 当日行うべき部位をホームに表示。2日以上休養済み=✓回復済、それ以外=⏳で色分け |
| 左右差リアルタイム表示 | weightL/R変更時に差異%をDOM直接更新。均等時は緑で⇔表示 |
| quickStartByPart() | おすすめカードから即セッション開始できる導線を実装 |
| レストポーズ法修正 | タイマー15秒→20秒、reps欄を空欄4スロット（漸減前提）に変更 |

| おすすめ判定修正 | `set.done===true` のセットが1つ以上ある場合のみ「完了済み」と判定 |
