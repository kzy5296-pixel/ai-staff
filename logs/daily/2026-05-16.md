## 今日学んだこと・気づき

1. **「楽天から仕入れてAmazonに出品しても採算が合わない」かどうかで判断する**
   - 楽天最安値 + FBA費用 > Amazon平均価格 なら、3P業者が参入しにくい → 価格回復しやすい
   - SAKANAQUARIUM限定盤：楽天¥9,477 vs Amazon avg¥8,647 → 採算NG → ◎

2. **「完全生産限定盤」は希少性が高く値崩れしにくい**
   - 増産されないため、Amazon在庫切れ後に価格が戻りやすい
   - 新品3P最高¥17,140（3月）という実績が根拠

3. **CERO-Zゲームは仕入れ前に確認必須**
   - CoD Black Ops7はAmazonページを開いたら即アダルト警告 → 事前にKeepaでジャンル確認

4. **入荷予定ありの商品は慎重に**
   - iwakiは「入荷予定あり」 → Amazonが継続して安値供給する可能性
   - 在庫切れが前提の刈り取りでは不利な条件

5. **靴系ブランドはスクリーニング段階でフィルタする**
   - アシックス・ニューバランス・メレル等は出品制限リスクが高い → 今後除外ワードに追加

6. **ゲーム限定版・スペシャルボックスも出品制限が多い**
   - 今回2件（五等分の花嫁・Redemption Reapers）が制限でNG
   - 限定版ゲームは条件が良くても制限確認を最優先に

7. **「残り1〜2点」は仕入れ候補から外す方が効率的**
   - 量が買えず利益インパクトが小さい
   - スクリーニング時に在庫数フィルターを追加検討
