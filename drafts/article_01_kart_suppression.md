## 1. はじめに：刈り取りで「価格が戻らない」失敗の正体

Amazon刈り取りを始めると、最初に必ずぶつかる壁があります。

> 「Amazon が大幅値下げ → 仕入れた → 在庫切れになった → なのに価格が戻らない！」

僕も最初これに何度もハマりました。Keepa で deltaPercent が 30〜40% の商品を見つけて買ったのに、Amazon の在庫が切れた後も他の3Pセラーが安値で出してきて、思ったように利益が乗らない…。

調べてみると、原因はシンプルでした。

**「他の3Pセラーが、楽天や Yahoo! ショッピングから安く仕入れて参入してくる商品だったから」**

つまり、Amazon が値下げしていても、**他のECサイトに安い在庫が残っている商品は刈り取りに向きません**。

---

## 2. 楽天採算NG構造とは

ここで使えるのが「**楽天採算NG構造**」という判断軸です。

定義はシンプル：

> 「楽天から仕入れて Amazon で売っても、3Pセラーが採算が合わない構造になっている商品」

具体的には、こういう状態を指します：

```
楽天の最安値 + FBA費用（送料・手数料） > Amazon の3P平均価格
```

つまり「楽天から仕入れたら赤字になる」状態。
こうなっていると、他の3Pセラーは楽天から仕入れて参入できない → Amazon 在庫切れ後に価格がしっかり回復する。

## 3. 実例：SAKANAQUARIUM 限定盤Blu-ray の仕入れ

実際に最近僕が仕入れた商品で説明します。

### 商品データ

| 項目 | 値 |
|---|---|
| 商品名 | SAKANAQUARIUM 2025 "怪獣" 完全生産限定盤Blu-ray |
| ASIN | B0G5NFW8FC |
| Amazon 仕入れ価格 | ¥6,040 |
| Keepa Amazon 90日平均 | ¥8,647 |
| Keepa 新品3P 90日平均 | ¥10,110 |
| Keepa 新品3P 最高値 | ¥17,140（直近3ヶ月） |
| **楽天市場の最安値** | **¥9,477** |
| ランキング | DVD部門 #123（超高回転） |
| 評価 | ★4.8（117件） |
| FBA 3P競合 | 1件のみ（仕入れ時点） |

### 判断

**楽天 ¥9,477 vs Amazon avg90 ¥8,647**

→ 楽天で仕入れて Amazon で売ろうとしても、楽天価格 > Amazon 平均価格 で赤字。
→ つまり「**楽天採算NG**」が成立。
→ 他の3Pセラーが楽天ルートで安値参入する余地がない → 価格回復が期待できる。

加えて：
- **「完全生産限定盤」**で増産されない（供給が今後増えない）
- **新品3P最高 ¥17,140** の実績あり（直近3ヶ月で）
- 評価★4.8で返品リスク低い

→ **¥6,040 × 3個 = ¥18,120 で仕入れ**

---

## 4. 「楽天採算NG」が大事な理由 ①：他セラーが参入しにくい

これは多くの人が理解している、刈り取りの基本原理です。

Amazon 在庫切れ後、価格が戻るかどうかは「**他の3Pセラーが安く出品してこないか**」で決まります。

楽天で安く買えるなら、誰でもそこから仕入れて Amazon に出品してきます。すると：
- 3P出品者が10人、20人と増える
- 価格競争で値下がりする
- カート争いに勝てない
- 結局、利益が出ない

楽天採算NG構造なら、この参入が起きにくい。これが第一の効果。

---

## 5. 「楽天採算NG」が大事な理由 ②：Amazonカート表示の仕組み

そして、最近僕が出品作業をしていて気づいたのが **2つ目の効果** です。

### 「Amazon内で最安値なのにカートに出ない」現象

SAKANAQUARIUM を仕入れて自己出品した後の話です。

- 出品価格：**¥9,600**（Amazon内 3P最安値）
- それなのに「カート」に出てこない
- 「すべての出品を見る」に進んでようやく1番上に出てくる

なんで？と調べたら、原因は楽天価格でした。

### 仕組み：Amazonは外部サイトの価格を監視している

Amazonは楽天・Yahoo! ショッピング・ヨドバシなど**外部ECサイトの価格を常にチェック**しています。
自分の出品価格が外部サイトの最安値より高い場合、**Featured Offer（カート）が抑制される**仕様。

今回のケース：

```
カズヤの出品価格 ¥9,600  >  楽天最安値 ¥9,477
                       → Amazon: 「ユーザーには楽天で買った方が得だから、カートは表示しない」
                       → Featured Offer 抑制
                       → 売れにくい
```

### 「楽天採算NG」だとカートが復活しやすい

逆に楽天が高い（採算NG構造）商品は：

```
カズヤの出品価格 ¥9,500  <  楽天最安値 ¥12,000
                       → Amazon: 「Amazonが一番安い」
                       → カート表示 OK
                       → 売れる
```

つまり「楽天採算NG」は、**仕入れ後の売りやすさにも直結している**ということ。

## 6. 仕入れ判断フローまとめ

僕が今使っている判断フローはこれです。

### 必須条件（全部✅じゃないと買わない）

1. ✅ Amazon 1P（販売元 Amazon.co.jp）が出品中
2. ✅ 現在価格が 90日平均より 20%以上 安い
3. ✅ 現在価格が 90日間の最安値（isLowest90）
4. ✅ **楽天・Yahoo! でも在庫がない or 大幅に高い**（楽天採算NG構造）← 超重要
5. ✅ 3P出品者が少ない（10件以下が目安）
6. ✅ 評価 ★4.0以上（返品リスク低減）
7. ✅ ランキング 全体5,000位以内
8. ✅ 小型〜中型（≤2kg、≤450mm）でFBA手数料を最小化

### 判断ワークフロー

```
Keepa Product Finder で候補抽出
        ↓
Amazon 商品ページで条件1〜3確認
        ↓
楽天・Yahoo! で在庫・価格確認 ← ここが運命の分かれ目
        ↓
楽天が高い / 在庫薄い  →  仕入れ候補
楽天が安い / 在庫厚い  →  即パス
```

## 7. まとめ

Amazon刈り取りで成功する商品の本質は：

> **「Amazon在庫切れ後に、他のセラーが楽天等から安く仕入れて参入できない商品」**

そしてこれは仕入れ判断だけじゃなく、**Amazon のカート表示の仕組みにも直結している**ので、必ずチェックする価値があります。
